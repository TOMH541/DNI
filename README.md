# CIU Utilities Bot

## Features
- Role Request System
- Approve / Deny Buttons
- Only Allowed Roles Can Approve
- Easy Command System
- DM Invite Command

## Setup

1. Install Node.js
2. Open terminal in bot folder
3. Run:

npm install

4. Fill out `.env`

5. Start the bot:

npm start

## Important

Inside `commands/rolerequest.js` replace:

ROLE_ID_HERE

with the role ID allowed to approve/deny requests.

## Commands

/rolerequest
/sendinvite
